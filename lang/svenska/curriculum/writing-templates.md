# Каркасы писем (kirjoittaminen)

YKI письмо: №1–3 короткие ответы на ситуацию (20–50 слов), №4–5 текст с мнением (~80–120 слов).
Зубрить скелет — openings/closings — работает везде. На уровне 3 структура + выполнение задачи
важнее идеальной грамматики.

## A. Неформальное письмо/мейл (другу, коллеге)

```
Hej [namn]!

[Tack för ditt mejl. / Vad roligt att höra från dig!]

[Основной текст: ответь на вопросы, расскажи, спроси...]

[Vad tycker du? / Hör av dig! / Vi hörs!]

Hälsningar,
[Ditt namn]
```
- Приветствие: **Hej!** (или *Tjena!* — совсем неформально).
- Прощание: **Hälsningar** (нейтрально), **Kram** (объятие — близкому), **Vi hörs!** (до связи).

## B. Формальное письмо (ведомство, фирма, учитель)

```
Hej,
[или: Bästa [namn], / Till [mottagare],]

[Anledning: Jag skriver för att ... / Jag vill gärre ...]

[Основной текст. Несколько вопросов → оформи по пунктам.]

[Tack på förhand! / Jag hör gärna från er.]

Med vänliga hälsningar,
[Ditt namn]
[Telefon / e-post]
```
- Приветствие: **Hej,** (нейтрально-вежливо), **Bästa [имя],** (уважительно).
- Прощание: **Med vänliga hälsningar,** ↔EN "Kind regards,".

## C. Короткое сообщение (ситуация, №1–3)

Формула: **реакция → короткий ответ → вопрос/предложение.**
```
Hej! [Tack! / Vad kul! / Vad tråkigt.]
[короткий ответ по ситуации]
[Vilken tid passar dig? / Ska vi ses kl. 18?]
Hälsningar, [namn]
```

## Полезные «клеящие» фразы (связность)
- *Först och främst ...* ↔EN "First of all..."
- *Dessutom ...* ↔EN "Besides / In addition..."
- *Istället ...* ↔EN "Instead..."
- *Å ena sidan ... å andra sidan ...* ↔EN "On one hand ... on the other hand..."
- *Sammanfattningsvis ...* ↔EN "In summary..."
- *Till exempel ...* ↔EN "For example..."
- *Jag tycker att ...* ↔EN "I think that..."

## Чеклист «выполнение задачи» (главное на уровне 3)
- [ ] Ответил **на все** части задания (сколько вопросов — столько ответов).
- [ ] Подобрал **регистр** (Hej vs Bästa) под получателя.
- [ ] Соблюдён **объём** (короткое ~20–50; мнение ~80–120 слов).
- [ ] Есть приветствие и прощание.
- [ ] Смысл ясен (грамматика может хромать — это не критично).
