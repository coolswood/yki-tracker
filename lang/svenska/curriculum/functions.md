# 12 коммуникативных функций + каркасы (cognate-теги)

Тест награждает за **функции**, а не за грамматику. ~12 функций × 2–3 фразы = собираешь почти
любой ответ из «лего». Каркасы зубрить целиком. Статус ведём в `progress/mastery.md`.

Легенда: ⚪ new · 🟡 learning · 🔵 review · 🟢 mastered

---

1. **Hälsa / presentera** (поздороваться / представиться) — ⚪
   - *Hej! Jag heter X. Jag kommer från Y.* ↔EN "Hi! My name is X. I come from Y."
   - *Trevligt att träffas.* ↔EN "Nice to meet you."

2. **Fråga** (спросить) — ⚪
   - *Vad heter du? / Var bor du? / Vad gör du på fritiden?*
   - *Kan du ...?* ↔EN "Can you ...?" (kan/can, без "to")

3. **Be om något** (попросить) — ⚪
   - *Kan du hjälpa mig?* ↔EN "Can you help me?"
   - *Skulle jag kunna få ...?* "Could I have/get ...?"

4. **Tacka** (поблагодарить) — ⚪
   - *Tack! / Tack så mycket!* ↔EN "Thanks! / Thanks so much!"
   - *Tack för hjälpen.* ↔EN "Thanks for the help."

5. **Hålla med** (согласиться) — ⚪
   - *Jag håller med.* ↔EN "I agree."
   - *Helt rätt! / Jag tycker också så.* "Exactly! / I think so too."

6. **Bjuda in** (пригласить) — ⚪
   - *Vill du komma på ...?* ↔EN "Do you want to come to ...?" (vill/will)
   - *Har du lust att ...?* "Do you feel like ...ing?"

7. **Tacka nej / avböja** (вежливо отказаться) — ⚪
   - *Tyvärr, jag kan inte.* ↔EN "Unfortunately, I can't." (kan/can)
   - *Det är jättesnällt, men jag har inte tid.* "That's very kind, but I don't have time."
   - *Tyvärr kan jag inte, för att ...* "...because..."

8. **Jämföra** (сравнить) — ⚪
   - *... är större än ...* ↔EN "... is bigger than ..." (än/than)
   - *Båda är bra, men ... är bättre.* "Both are good, but ... is better."

9. **Uttrycka åsikt** (выразить мнение) — ⚪
   - *Jag tycker att ...* ↔EN "I think that..." (tycker ≈ think)
   - *Enligt mig ...* ↔EN "In my opinion..." (enligt/according)
   - *För det första ... för det andra ...* ↔EN "Firstly... secondly..."

10. **Ursäkta** (извиниться) — ⚪
    - *Ursäkta! / Förlåt!* ↔EN "Excuse me! / Sorry!" (förlåt/forgive)
    - *Jag är ledsen att jag är sen.* "I'm sorry I'm late."

11. **Föreslå** (предложить) — ⚪
    - *Vi kan ...* ↔EN "We can..."
    - *Ska vi ...?* ↔EN "Shall we...?" (ska/shall)
    - *Vad tycker du om att ...?* "What do you think about ...ing?"

12. **Avboka / ändra** (отменить / изменить планы) — ⚪
    - *Jag måste tyvärr avboka.* "I unfortunately have to cancel."
    - *Kan vi flytta vår tid?* "Can we move our appointment?"

---
**Принцип сборки:** задача YKI = обычно 2–3 функции подряд.
*Пример:* пригласить (#6) → получить отказ (#7) → предложить альтернативу (#11).
