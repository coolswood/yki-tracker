# План подготовки (финский) — горизонт ~1 год, буфер до 2

Цель: быть готовым к ~6–8 мес., сдать как только метрики позеленеют. Темп устойчивый,
без выгорания (30 мин/день). Минимум усилий → надёжная **3**.

## Фазы

### Фаза 0 — Onboarding (сессия 1)
Диагностика: 3 мини-промпта (2 письма, 1 говорение). Ставим baseline в `progress/profile.md`
и `dashboard/data.js`. Уточняем известные языки для стратегии переноса.
Критерий выхода: есть baseline-уровень.

### Фаза 1 — Perus-suomi (Мес 1–2)
- Ставим ~12 функций-каркасов (`curriculum/functions.md`).
- Базовая лексика по темам (`curriculum/themes.md`).
- Грамматика high-yield: vokaaliharmonia, базовые sijamuodot, verbityypit 1–4,
  astevaihtelu, kieltosana (`curriculum/grammar-essentials.md`).
- Speaking confidence через заученные чанки (`curriculum/speaking-toolkit.md`).
- **Выход:** собираешь любой короткий ответ из каркасов; говоришь без ступора.

### Фаза 2 — Aihe-mestaruus (Мес 3–5)
- Проходим 10 тем. На каждой: кластер слов + 2 письма + 2 говорения.
- **Выход:** все 10 тем тронуты, несколько 🟢.

### Фаза 3 — Simulation & hiominen (Мес 6–8)
- Прогоны в темпе экзамена. Ремонт слабых тем. Регистр/формальность.
- **Выход:** устойчивая 3 с заходом в 4. → можно сдавать.

### Фаза 4 — Mock & loppusuora (Мес 9)
- Полные mock-тесты в реальных условиях. Добивка пробелов.

## Ритм
- До 30 мин/день. Лёгкий режим («полегче») = только повторение + 1 лёгкий промпт.
- Streak не должен рваться — это мягкое давление для мотивации.

## Метрики готовности к сдаче
- Уровень ≥ 3.0 стабильно (≥5 сессий подряд) в `assessment.md`.
- ≥ 6/10 тем 🟢, все 10 начаты.
- Mock-тест письменно пройден без критичных ошибок (выполнение задачи 100%).
