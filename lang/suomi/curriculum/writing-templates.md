# Каркасы писем — kirjoittaminen (финский)

> YKI письмо: №1–3 короткие ответы на ситуацию (20–50 слов), №4–5 текст с мнением (~80–120 слов).
> Зубрить скелет (приветствия/прощания) работает везде. На уровне 3 структура + выполнение задачи
> важнее идеальной грамматики.
>
> Это **скелет**: структура каркасов зафиксирована, финские формулировки = TODO.

## A. Неформальное письмо/мейл (другу, коллеге)
```
Hei [nimi]!

[TODO: реакция на письмо]

[Основной текст: ответь на вопросы, расскажи, спроси...]

[TODO: закрытие / встречный вопрос]

[Tervehdys / Ystävällisin terveisin,]
[nimesi]
```
- Приветствие: **Hei!** (нейтрально), _TODO: неформальные варианты_.
- Прощание: **Ystävällisin terveisin** (нейтрально), _TODO: варианты_.

## B. Формальное письмо (ведомство, фирма, учитель)
```
Hei,
[tai: Hyvä [nimi], / [vastaanottaja],]

[TODO: причина: Kirjoitan siksi, että ...]

[Основной текст. Несколько вопросов → оформи по пунктам.]

[TODO: Kiitos etukäteen! / Odotan vastaustanne.]

[Ystävällisin terveisin,]
[nimesi]
[puhelin / sähköposti]
```

## C. Короткое сообщение (ситуация, №1–3)
Формула: **реакция → короткий ответ → вопрос/предложение.**
```
Hei! [TODO: Kiitos! / Kuinka hauskaa! / Kuinka ikävää.]
[короткий ответ по ситуации]
[TODO: Mihin aikaan sopii? / Nähdäänkö klo 18?]
[Terveisin, nimi]
```

## Чеклист «выполнение задачи» (главное на уровне 3)
- [ ] Ответил **на все** части задания.
- [ ] Подобрал **регистр** под получателя.
- [ ] Соблюдён **объём** (короткое ~20–50; мнение ~80–120 слов).
- [ ] Есть приветствие и прощание.
- [ ] Смысл ясен (грамматика может хромать — это не критично).
