# 12 коммуникативных функций + каркасы (финский)

> Тест награждает за **функции**, а не за грамматику. ~12 функций × 2–3 фразы = собираешь почти
> любой ответ из «лего». Каркасы зубрить целиком. Статус ведём в `progress/mastery.md`.
>
> Это **скелет**: 12 функций зафиксированы (те же концепты, что в шведской дорожке),
> финские фразы = TODO, наполняются на сессиях.

Легенда: ⚪ new · 🟡 learning · 🔵 review · 🟢 mastered

---

1. **Tervehtiä / esitellä itsensä** (поздороваться / представиться) — ⚪
   - _TODO: Hei! Olen X. ..._

2. **Kysyä** (спросить) — ⚪
   - _TODO: Mikä sinun nimesi on? / Missä asut? / Mitä teet vapaa-ajalla?_

3. **Pyytää** (попросить) — ⚪
   - _TODO: Voitko auttaa minua? / Voisinko saada ...?_

4. **Kiittää** (поблагодарить) — ⚪
   - _TODO: Kiitos! / Kiitos paljon! / Kiitos avusta._

5. **Olla samaa mieltä** (согласиться) — ⚪
   - _TODO: Olen samaa mieltä. / Aivan oikein!_

6. **Kutsua** (пригласить) — ⚪
   - _TODO: Haluatko tulla ...?_

7. **Kieltäytyä kohteliaasti** (вежливо отказаться) — ⚪
   - _TODO: Valitettavasti en voi. / Se on todella ystävällistä, mutta ..._

8. **Vertailla** (сравнить) — ⚪
   - _TODO: ... on suurempi kuin ... / molemmat ovat hyviä, mutta ..._

9. **Esittää mielipide** (выразить мнение) — ⚪
   - _TODO: Mielestäni ... / Ensinnäkin ... toiseksi ..._

10. **Pyytää anteeksi** (извиниться) — ⚪
    - _TODO: Anteeksi! / Olen pahoillani, että ..._

11. **Ehdottaa** (предложить) — ⚪
    - _TODO: Me voimme ... / Ehdotan, että ..._

12. **Perua / muuttaa suunnitelmia** (отменить / изменить планы) — ⚪
    - _TODO: Minun täytyy valitettavasti perua. / Voimmeko siirtää ajan?_

---
**Принцип сборки:** задача YKI = обычно 2–3 функции подряд.
*Пример:* пригласить (#6) → получить отказ (#7) → предложить альтернативу (#11).
