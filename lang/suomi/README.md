# YKI Suomi — дорожка подготовки (финский)

Подготовка к **YKI Finnish** (keskitaso), фокус на **kirjoittaminen** (письмо) и
**puhuminen** (говорение). Цель: надёжная **3** с минимумом усилий.

> Это языковая дорожка внутри общего мультиязычного трекера. Общий протокол сессий и
> движок дашборда живут в `core/` (в корне репо). Здесь — только финский контент.

## Что внутри (`lang/suomi/`)
```
lang/suomi/
├── config/student.md          параметры студентки
├── PLAN.md                    фазы и таймлайн
├── curriculum/                опорные материалы
│   ├── grammar-essentials.md  грамматика (vokaaliharmonia, sijamuodot, verbityypit, astevaihtelu)
│   ├── themes.md              10 тем YKI + кластеры слов
│   ├── functions.md           12 коммуникативных функций + каркасы
│   ├── writing-templates.md   каркасы писем (формал./неформал.)
│   └── speaking-toolkit.md    маркеры, филлеры, фразы взаимодействия
├── bank/                      банк заданий в стиле YKI
│   ├── writing-prompts.md
│   └── speaking-prompts.md
├── progress/                  ПАМЯТЬ о студентке
│   ├── profile.md             снимок уровня
│   ├── mastery.md             матрица тема×функция + даты SR
│   ├── mistakes.md            лог ошибок по паттернам
│   └── assessment.md          скользящие оценки 1–6
├── sessions/log.md            история сессий
├── reviews/due.md             очередь повторения на сегодня
└── dashboard/
    ├── index.html             открыть в браузере
    └── data.js                данные (обновляются после сессии)
```

> В настоящий момент — стартовый **скелет**: структуры зафиксированы, лексика/примеры
> помечены TODO и наполняются по мере сессий.

## Как заниматься
1. Сказать ассистенту «занимаемся финским» / «сессия suomi».
2. Ассистент читает `core/AGENTS.md` (протокол) + память здесь (`progress/`), даёт промпт в стиле YKI.
3. Отвечаешь по-фински → исправления + правило.
4. В конце — оценка и обновление всех файлов + `dashboard/data.js`.
5. Открыть `dashboard/index.html` (или F5) — увидеть прогресс.
