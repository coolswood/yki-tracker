# Сессия 1 — 2026-06-25 (диагностика + режим переводов)

**Фокус:** диагностика уровня, переход в режим «перевод по предложениям».
**Режим:** студент попросил маленькие шаги (перевод RU→SV по предложениям, повторение до автоматизма).
**Оценка сессии:** 2.5/6 (письмо-короткие) — стартовый baseline. Конструкции осваиваются быстро.

## Что пройдено (доведено до автоматизма)
1. **Знакомство:** `Jag heter X. Jag bor i Y.` (heter, bor i)
2. **Профессия:** `Jag jobbar/arbetar som + profession` (som = «as»)
3. **Отрицание:** `inte`-placement → `subjekt + VERB + inte` (освоено на 2 глаголах)
4. **Вопросы:** `frågeord + verb + subjekt` (без «do») → `Var bor du?`
5. **Тема Familj:** `Jag har en/ett …` (har = have); множ. число — паттерн `-ar` для en-слов и «без изменения» для ett-слов (barn, hus).
6. **Связный мини-текст:** студент собрал 3-фразовый рассказ о себе (ступень 2 лестницы).

## Промпты и ответы студента

**D1 (письмо, короткое):** граньна приглашает на ужин → согласись, спроси что принести.
- Студент: *Hej! Tack för inbjuden! Jag ska glad att komma till ditt hus! Vad ska jag ta?*
- Правки: inbjuden→**inbjudan**; *Jag ska glad*→*Jag blir glad / ska bli glad*; *ditt hus*→*din middag*; *ta*→*ta med*.
- Оценка: 3/6 (задача выполнена, смысл ясен).

**D2 (письмо, короткое):** коллега болен, передай бумаги.
- Студент: *Hej! Det är veldigt ledsen att du är sjuk! Ja, jag kan ta papper på kontoret! Jag önskar du att vara frisk.*
- Правки: veldigt→**väldigt**; *Det är ledsen*→*Det är ledsamt / Vad tråkigt att...*; *Ja, jag kan ta*→*Självklart kan jag lämna*; *papper*→*papperna*; *Jag önskar du att vara frisk*→*Hoppas du blir frisk snart! / Krya på dig!*
- Оценка: 2.5/6 (задача выполнена, больше грамм. ошибок, неестественная формулировка).

**Режим переводов (мини-предложения):**
1. «Меня зовут Анна. Я живу в Хельсинки.» → *Jag heter Anna. Jag bor i Helsingfors.* ✅ (опечатка Helsingforst)
2. «Я работаю учителем.» → *Jag jobbar lärare* → ✅ *Jag jobbar **som** lärare* (пропуск `som`)
3. «Я работаю врачом.» → *Jag jobbar som läkare.* ✅
4. «Она работает инженером.» → *Hon arbetar som ingenjör.* ✅
5. «Я не работаю учителем.» → *Jag jobbar som inte lärare* → ✅ *Jag jobbar **inte** som lärare* (`inte`-position)
6. «Я не работаю врачом.» → *Jag jobbar inte som läkare.* ✅
7. «Я не живу в Хельсинки.» → *Jag bor inte i Helsingfors.* ✅ (опечатка Helsigfors)
8. «Где ты живёшь?» → *Var bor du?* ✅

## Тема Familj + множ. число (продолжение сессии)
9. «У меня есть сестра.» → *Jag har en syster.* ✅

12. ett-слово без изменения: «У меня трое детей.» → *Jag har tre barn.* ✅
13. Связный текст: «Меня зовут… живу… работаю… двое детей» → *Jag heter Daniil. Jag bor i Helsingfors och arbetar som lärare. Jag har två barn.* ✅ (только знак `<` вместо точки)

## Зафиксированные паттерны ошибок
- `som-as` — пропуск `som` в `jobba som + профессия` (1 раз, затем исправлено → автоматизм).
- `inte-position` — `inte` не сразу после глагола (1 раз, затем исправлено → автоматизм на 2 глаголах).
- `plural` — множ. число (зона роста: пришёл верно со подсказки, но самому пока трудно). ⚠ следить
- `punctuation` — знак `<` вместо точки в конце (опечатка, 1 раз).
- `lex` — inbjuden/inbjudan; ta vs ta med; ta vs lämna; önskar вместо hoppas/krya.
- `adjective-kongruens` — ledsen vs ledsamt (с det).
- `definite` — papper → papperna.
- `modal-vocab` — ska + прилагательное (нужен глагол).

## Что хорошо
- Быстро схватывает конструкции после 1 исправления → повторяет верно.
- Когнаты работают (jobbar, lärare, ingenjör, kontor) — узнавание мгновенное.
- Структура ответов логичная (реакция → действие → пожелание).

## Продолжение сессии (вечерний заход): множ. число — ворота + прошедшее время

### Ворота мастерства: множественное число
Челлендж 10 предложений: 7/10 чисто. Ошибки: böker→böcker (повтор), läsar→läser, *två tidning* (забыл -ar), *veckor* (не знал -or-паттерн, подсмотрел).
→ тема НЕ пройдена, крутили слабые места: 3 добора (böcker, tidningar, veckor) — все ✅.
→ **мн. число закрыто** (все 3 варианта: -ar / -er (böcker) / -or (veckor) / ett без изменения).

### Новая тема: прошедшее время (dåtid)
Паттерн = «английский -ed ≈ шведский -de». Три варианта регулярного прошедшего:
1. «Я работал учителем.» → *Jag arbetade **om** lärare* → ✅ *Jag arbetade **som** lärare* (past верно — arbetade; оговорка om вместо som).
2. «Я работал врачом.» → *Jag arbetade som läkare.* ✅
3. «Я читал книгу.» → *Jag läsede en bok* → сам поправил → ✅ *Jag läste en bok* (över-regularisering läsede).
4. «Я жил в Стокгольме.» → *Jag bodde i Stockholm.* ✅ (bo→bodde, -dde)
5. Капка (интеграция): «Я работал учителем и читал две книги.» → *Jag jobbade som lärare och läste två böcker.* ✅ (past + som + plural вместе)

Регрессии/оговорки на ранее освоенном:
- frågeordning: *Var du arbetar* → ✅ *Var arbetar du* (заминка, исправлено).
- som: *arbetade om lärare* → om вместо som (оговорка, исправлено).
